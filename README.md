flummox-boilerplate
===================

This is my application's boilerplate for something. It's ready to use:

- gulp
- babelify
- browserify
- vinyl-source-stream
- react
- flummox

Getting Started
-------------

    $ npm install
    $ gulp

memo
----

    $ npm install --save react flummox
    $ npm install --save-dev babelify browserify gulp vinyl-source-stream
